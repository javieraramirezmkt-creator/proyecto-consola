# Aplicación de Consola - Fundamentos JavaScript

## Descripción
Aplicación desarrollada en JavaScript que permite realizar operaciones matemáticas básicas e implementar estructuras fundamentales del lenguaje.

## Funcionalidades

- Operaciones matemáticas (suma, resta, multiplicación, división)
- Uso de condicionales (if, switch)
- Uso de ciclos (for, while)
- Manipulación de arreglos
- Uso de objetos
- Modularización mediante funciones
- Validación de datos del usuario

## Tecnologías utilizadas

- JavaScript
- HTML5

## Cómo ejecutar

1. Abrir el archivo `index.html`
2. Abrir la consola del navegador (F12)
3. Interactuar con los prompts
